## Python Implementierung

Die Python Implementierung besteht aus einem kurzen Script. Dieses enthält eine Klasse, die die Datenstruktur definiert, den Algorithmus und eine Mainfunktion, die beim ausführen des Scripts abläuft.
JSON Deserialisierung ist in Python Bestandteil der Standardbibliothek, so dass keine externen Abhängigkeiten benötigt werden.
